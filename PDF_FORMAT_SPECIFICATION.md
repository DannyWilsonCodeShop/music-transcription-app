## Overview
ChordScout generates professional PDFs with lyrics, chords, and Nashville Number System notation for musicians.

## PDF Layout Structure

### Header Section
- **Song Title** (20pt, bold)
- **Key Signature** (14pt, bold) - e.g., "Key: Bm minor"

### Content Sections
For each verse/chorus/section:

1. **Lyrics** (12pt, normal) - displayed first
2. **Staff Lines** - 5 horizontal lines for musical notation
3. **Chord Names** (10pt, bold, blue) - above staff
4. **Nashville Numbers** (10pt, bold, red) - below staff

### Footer
- "Generated by ChordScout - Nashville Number System" (8pt, gray)

## Nashville Number System Rules

### Basic Numbers (Major Key)
- **1** = Tonic (key center)
- **2** = Supertonic  
- **3** = Mediant
- **4** = Subdominant
- **5** = Dominant
- **6** = Submediant
- **7** = Leading tone

### Accidentals
- **b2, b3, b5, b6, b7** = Flat intervals
- **#1, #2, #4, #5, #6** = Sharp intervals

### Chord Quality Indicators
- **Lowercase** = Minor chord (e.g., "6" = Am in C major)
- **7** = Dominant 7th (e.g., "57" = G7)
- **maj7** = Major 7th (e.g., "1maj7" = Cmaj7)
- **Â°** = Diminished (e.g., "7Â°" = BÂ°)
- **+** = Augmented (e.g., "1+" = C+)

2. **Easy Transposition** - Numbers stay the same when changing keys

## Technical Implementation

The PDF generator:
- Calculates intervals between chord roots and key center
- Maps intervals to Nashville numbers using semitone math
- Handles enharmonic equivalents (C# = Db)
- Supports both major and minor keys
- Adds chord quality indicators automatically
- Draws staff lines with proper spacing
- Positions chords and numbers precisely
